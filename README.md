SauceDemo Cypress Automation 
Overview
This project automates UI testing for the SauceDemo website using Cypress.

Learning Objectives
Learn Cypress framework basics
Implement Page Object Model (POM)
Use reusable test steps via custom commands
Features Implemented
✅ Login failure validation
✅ Successful login and homepage check
✅ Product navigation and add-to-cart validation
✅ Custom commands for reusable steps
✅ Page Object Model structure
